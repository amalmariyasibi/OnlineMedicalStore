## 🎨 UI Elements Explained

### Medicine Cards
```
┌─────────────────────────────────────────────────┐
│ Medicine Name (Blue, Bold)     [View in Store]  │
│ Description text (Gray)                         │
│ ℹ️ Dosage: Detailed dosage info (Small text)   │
└─────────────────────────────────────────────────┘
```

### Urgency Indicators
```
🔴 High Urgency (Red background)
🟡 Moderate Urgency (Yellow background)
🟢 Low Urgency (Green background)
```

### Message Types
```
User Messages: Blue background, right-aligned
Bot Messages: Gray background, left-aligned
```

### Icons Used
- 💊 Medicine/Symptoms
- 🏥 Doctor consultation
- 💬 General chat
- 💡 Tips and advice
- ⚠️ Disclaimer
- 🚨 Emergency alert
- ℹ️ Information
- 🤖 Bot indicator

## 📱 Mobile View

On mobile devices, the layout adapts:
- Full-width messages
- Stacked medicine cards
- Touch-friendly buttons
- Scrollable content
- Fixed input at bottom

---

## 🎯 Key Visual Features

1. **Clean Layout**: Organized, easy to read
2. **Color Coding**: Different colors for urgency levels
3. **Icons**: Visual cues for different sections
4. **Professional Cards**: Medicine information in structured cards
5. **Clear Hierarchy**: Important info stands out
6. **Responsive**: Works on all screen sizes
7. **Accessible**: High contrast, readable fonts
